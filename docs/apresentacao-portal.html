Portal / Dashboard
Acesso centralizado a todos os sistemas
Módulo Principal
O Portal é o ponto central de acesso a todos os sistemas WKL. Após o login — autenticado localmente ou via Active Directory — o usuário visualiza o dashboard com todos os módulos disponíveis conforme seu perfil, além de notificações e atalhos rápidos.
Funcionalidades Principais
Autenticação dupla — Login local (bcrypt) ou via Active Directory (LDAP)
Dashboard — Cards de acesso rápido a todos os módulos
Gestão de usuários — Criar, editar e desativar usuários locais ou AD
Configurações — SMTP, Telegram, integrações pelo painel admin
Visibilidade — Exibir ou ocultar módulos por nível de usuário
Log de atividades — Auditoria completa de todas as ações
Menu responsivo — Navegação lateral com todos os módulos
API Key — Autenticação server-to-server via x-api-key
Active Directory — SSO com domínio Windows via LDAP
