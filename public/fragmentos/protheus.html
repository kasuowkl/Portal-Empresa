Protheus

Configure a base da integração com o TOTVS Protheus e prepare os módulos de Patrimônio e Agenda Financeira.

Configuração geral do ProtheusDefina a URL base, autenticação, ambiente, empresa/filial e as opções do teste de conexão.
Desativado
Esta primeira fase cria a base do conector. Você já consegue salvar parâmetros, testar a conectividade e mapear módulos antes de ligar sincronizações reais.
Serviço
URL do serviçoExemplo: http://servidor-protheus:8080/rest
AmbienteNome do ambiente ou appserver exposto pelo Protheus
EmpresaCódigo da empresa utilizado na integração
FilialCódigo da filial principal para a integração
Autenticação
UsuárioUsuário técnico ou credencial usada pelo conector
Senha / tokenSenha do usuário, API Key ou token Bearer, conforme o tipo abaixo
Tipo de autenticaçãoBasic, API Key ou Bearer
Nome do header da API KeyUsado quando o tipo é API Key
Teste e monitoramento
Rota de health/checkEndpoint usado no botão de teste. Ex.: /health, /status
Timeout (ms)Tempo máximo do teste antes de considerar falha
Registrar logs técnicosGrava os testes e ações de integração nos logs do Portal
Ativado
Execução manual habilitadaPermite acionar integrações e testes manualmente pelo painel
Ativado
ObservaçõesCampo livre para documentar regras, rotas ou parâmetros combinados com o time do Protheus
Integração Protheus - PatrimônioDefina quando e como o Portal conversa com o Protheus para bens patrimoniais.
Desativado
Esta primeira fase prepara a integração do cadastro e das movimentações patrimoniais.
Eventos de Patrimônio
Novo bem cadastradoEnvia um novo patrimônio do Portal para o Protheus
Transferência de bemAtualiza responsável, centro de custo, setor ou localização no Protheus
Baixa / descarteMarca o ativo como baixado ou inativo no Protheus
Mapeamentos
Endpoint de patrimônioRota REST principal do Protheus para ativos
Alias / tabelaEx.: SN1, BEM, ATIVO
Campo do código do bemIdentificador do patrimônio no Protheus
Campo do centro de custoUsado em transferências e relacionamento com setores
Campo da situação/statusCampo usado para baixa, manutenção ou ativo/inativo
Integração Protheus - Agenda FinanceiraConfigure a exportação de contas, pagamentos e conciliações do Portal para o financeiro do Protheus.
Desativado
Use esta aba para preparar o envio de lançamentos da Agenda Financeira ao Protheus.
Eventos Financeiros
Nova conta criadaExporta um lançamento financeiro do Portal para o Protheus
Conta pagaAtualiza o título/baixa financeira no Protheus quando o Portal marcar como pago
Consultar status no ProtheusPermite atualizar no Portal o retorno de baixa ou processamento vindo do Protheus
Mapeamentos
Endpoint financeiroRota REST principal para contas/títulos
Prefixo de documentoPrefixo usado ao gerar números únicos para o Protheus
Natureza padrãoNatureza / tipo financeiro padrão da integração
Campo de vencimentoCampo do Protheus que recebe a data de vencimento
Campo de valorCampo do Protheus que recebe o valor da conta
Campo do fornecedor / clienteCampo usado para vínculo com empresa/fornecedor
